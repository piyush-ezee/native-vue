# native-vue


### Add Platform
```
ns platform add android
```

### Run App with Debug Mode
```
ns run android --debug
```

### Build Android APK
```
ns build android
```
> Locate generated APK in Path shown in output (e.g. platforms/android/app/build/outputs/apk/debug/app-debug.apk) and rename it.

### Clean Platform Data
```
ns clean
```
